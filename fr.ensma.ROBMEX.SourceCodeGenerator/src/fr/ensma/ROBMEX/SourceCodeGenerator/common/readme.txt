	Project: ROBMEX - Templates for generation source code
	
	---
	
	generateCMakeLists: attempt to generate an important file for compiling the ROS system (CMakeLists.txt)
	generateMission: attempt to describe the generation of source code for any instance of ROSMiLan
	generatePackageXML: attempt to generate an important file for compiling the ROS system (package.xml)
	generateRosNode: attempt to describe the generation of source code specifically for a ROS node
	generateRosPackage: attempt to generate every important file that compose a ROS package, including the node but also message structures, folders, etc.

	Generated files are found in src-gen
